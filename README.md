# 📜 Skript Utilities

A collection of useful and lightweight **Minecraft Skripts** designed to improve server functionality and staff tools.

---

## ✨ Included Skripts

### 💬 PlayerCommands

Simple utility commands for players that every server needs.

**Features:**

* `/discord`, `/dc`, `/disc` → Sends Discord invite link
* `/rules` → Displays server rules

---

### 🛡️ ScreenShare System

Basic staff tool for handling player screenshares.

**Features:**

* `/screenshare <player>` → Teleports staff & jails player
* `/release <player>` → Releases player and restores state
* Prevents double screensharing

**Permissions:**

* `ss.use`
* `ss.release`

---

### 📢 Broadcast System

Flexible broadcast commands for announcements.

**Features:**

* `/bc <message>` → Broadcast message
* `/broadcast <message>` → Alternative command
* `/bctitle <message>` → Sends title + chat broadcast

**Permissions:**

* `servername.bc`          *Put your servername here!*
* `servername.broadcast`   *Put your servername here!*
* `servername.bctittle`    *Put your servername here!*

## ⚙️ Installation

1. Install **Skript** on your server
2. Place `.sk` files into:
   `/plugins/Skript/scripts/`
3. Run:

```
/sk reload all
```
